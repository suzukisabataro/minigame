# minigame
javascriptでRPGを作成しました。

## Description
javascriptでRPGを作成。
条件分岐により簡単なイベント、戦闘を実装しました。

***DEMO:***
https://gyazo.com/a0862de0d6702aa559739d9c943b8cf4

## Features
- ランダムエンカウントで戦闘発生
- 戦闘後経験値獲得でレベルアップ
- 建物接触でHP回復
